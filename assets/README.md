# Assets Folder

## Montessori Avatar Setup

To use a custom Montessori avatar in the student chat:

1. Upload your Montessori avatar image here as: `montessori-avatar.png`
2. The image should be:
   - Square ratio (e.g., 128x128px or 256x256px)
   - PNG format with transparent background recommended
   - Represents Montessori's cosmic education philosophy

Currently using: 🤖 emoji as placeholder

Once uploaded, the avatar will be accessible at: `/assets/montessori-avatar.png`
