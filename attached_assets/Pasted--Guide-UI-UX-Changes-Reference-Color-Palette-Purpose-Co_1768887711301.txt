Color Palette
Purpose	Color	Usage
Primary Accent	#789A76	Links, highlights, accent text
Hover Accent	#5a7a58	Hover states
Cream Background	#FAF9F6	Card backgrounds
Text Primary	#2E2E2B	Body text
Subtle Background	rgba(120, 154, 118, 0.08)	Light hovers
Medium Background	rgba(120, 154, 118, 0.12)	Active hovers
Soft Shadow	rgba(46, 46, 43, 0.04)	Card shadows
IMPORTANT: Zero dark green (#3d5a3d) anywhere.

Streamlit CSS Targeting Pattern
Since Streamlit doesn't support adding classes directly to widgets:

st.markdown('<div class="my-custom-class">', unsafe_allow_html=True)
# Streamlit widget here
st.markdown('</div>', unsafe_allow_html=True)
Then target with: .my-custom-class .stButton > button { /* styles */ }

Note: :has() and :contains() selectors don't work in Streamlit.
