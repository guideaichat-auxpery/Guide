Quick “Scroll to Top” Button

HTML:

<button id="scrollTopBtn" style="position:fixed;bottom:20px;right:20px;display:none;">⬆️ Top</button>


JavaScript:

const scrollBtn = document.getElementById('scrollTopBtn');

// Show button when scrolled down
window.onscroll = function() {
    if (document.body.scrollTop > 100 || document.documentElement.scrollTop > 100) {
        scrollBtn.style.display = "block";
    } else {
        scrollBtn.style.display = "none";
    }
};

// Scroll to top on click
scrollBtn.addEventListener('click', () => {
    window.scrollTo({
        top: 0,
        behavior: 'smooth'
    });
});


✅ This ensures a button appears after scrolling and animates a smooth scroll to the top.

2️⃣ Landing at the Top of the Page on Login

Sometimes after a login or page reload, the browser keeps the previous scroll position. To force it to top on load, use:

window.addEventListener('load', () => {
    window.scrollTo({
        top: 0,
        behavior: 'instant' // can also be 'auto'
    });
});


If you’re using a framework (React, Vue, etc.) inside Replit, the same logic should run after your component mounts.

3️⃣ Smooth Scroll to Anchors / Sections

If your page has links like:

<a href="#function1">Go to Function 1</a>


And your target section:

<div id="function1">
  <h2>Function 1</h2>
  <p>Details...</p>
</div>


Add smooth scrolling behavior:

document.querySelectorAll('a[href^="#"]').forEach(anchor => {
    anchor.addEventListener('click', function (e) {
        e.preventDefault();
        const target = document.querySelector(this.getAttribute('href'));
        if (target) {
            target.scrollIntoView({ behavior: 'smooth', block: 'start' });
        }
    });
});


✅ This ensures that clicking any anchor link scrolls smoothly to the appropriate section.

⚡ Quick Notes / Gotchas:

If your page is dynamically loaded (like using innerHTML or AJAX), make sure the scroll logic runs after the content exists.

For SPA frameworks (React/Vue/Svelte), you might need useEffect or onMounted hooks to handle scroll.

Some browsers on page reload may require history.scrollRestoration = 'manual' to prevent auto scroll to last position:

if ('scrollRestoration' in history) {
    history.scrollRestoration = 'manual';
}
